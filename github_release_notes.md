## SRM Client v1.0.0 🚀

### ✨ New Features
- 🔐 **Biometric Login**: Unlock with fingerprint or face ID
- ⚡ **Offline Cache**: View attendance, marks, timetable without internet
- 🎨 **Material UI**: Dynamic theming matches your phone's wallpaper
- 🌓 **Dark Mode**: Auto-switches based on system preference

### 🔒 Security
- Passwords are **never stored** or logged
- All data is encrypted using native `flutter_secure_storage`
- No telemetry or tracking included

### 📥 Installation
1. Download `app-release.apk`
2. If prompted, select "Install from unknown sources"
3. Launch, enter your NetID/password once, and use Fingerprint going forward!
